## 1. 요구사항

#### 1. 프로젝트 구조

> game/은 game을 하는 디렉토리입니다.
>
> ssafy5_3/은 학생 정보, 메세지 등이 있는 디렉토리입니다.

```bash
$ django-admin startproject rollingpaper .
$ python manage.py startapp game
$ python manage.py startapp ssafy5_3
```
